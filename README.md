# Artifacts Folder

This folder contains data files used in the project:

Feature_Categories.csv | Feature_Categories.json | Feature_Categories.yaml

Categorized raw data file with:
student info under Students, 
course info under Course_Info, 
activities/logins of the students under Engagement, 
grades/assessments of the students under Performance, 
and the rest like device or internet quality under Other.
